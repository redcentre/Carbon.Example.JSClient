Carbon HTML5 Demo Service Address
This application communicates with the Carbon Web Service to implement a basic cross-tabulation reporting system. The app is implemented using plain HTML5 techniques (html + styles + JavaScript) and is a part of the example projects and scripts available on Red Centre Software's GitHub page. Developers can evolve this example into a more sophisticated application, perhaps using one of the popular JS frameworks.
